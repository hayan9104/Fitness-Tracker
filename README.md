# Fitness Tracker

The **Fitness Tracker** is a web application designed to help users monitor and enhance their fitness journey by tracking various health metrics and activities.(Pg1.html is main page)

## Features

- **User Authentication**
- **Profile Management**
- **Activity Logging**
- **Progress Visualization**
- **Responsive Design**
